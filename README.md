# 🏏 IPL 2024 Players & Teams Analysis | Power BI Project

This project presents a comprehensive **data analysis and visualization of the IPL 2024 season**, focusing on **player performances** and **team statistics** using Microsoft Power BI. The analysis is based on ball-by-ball data extracted from the `ipl_2024_dataset`.

## 📊 Dashboard Highlights

The Power BI report includes the following key visualizations:

### 📌 Player Performance Analysis:
- Top Run Scorers of IPL 2024
- Best Strike Rate (Min 50 Balls Faced)
- Most Wickets Taken
- Best Bowling Economy (Min 10 Overs Bowled)
- Player-wise Runs and Wickets Breakdown
- Runs by Batsman by Match

### 📌 Team Performance Analysis:
- Match-wise Team Wins
- Team Win Percentage
- Toss Decision Trends
- Runs by Team per Match
- Highest Scoring Team & Match
- Powerplay Performance Comparison
- Death Overs Performance (16–20 Overs)

---

## 🧠 Key Insights

- **Top Performer**: [Player Name] scored the highest total runs in IPL 2024.
- **Bowling Star**: [Bowler Name] had the best economy rate among bowlers who bowled at least 10 overs.
- **Most Dominant Team**: [Team Name] had the highest win percentage.
- **Close Encounters**: Multiple matches were won with margins less than 5 runs or 1 wicket!

## 📥 How to Use

1. Download or clone this repository.
2. Open the `.pbix` file in **Power BI Desktop**.
3. Interact with filters (team, player, match) to explore the data.
4. Explore the DAX formulas and visuals to understand the logic.
